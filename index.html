Introduction

This webpage is a collection of reflections and summaries of my first coop workterm with the University of Guelph on the Communications and Public Affairs Division as a Web Developer and Data Analyst over the Fall 2021 semester. Presented is information pretaining to my employer, different learning goals I had, and summaries of what I worked on and skills I learned.

Employer Information

This work term I had the pleasure to work with the University's Web Strategy Manager, James Harris, on the Communications and Public Affairs team where I spent my time working on various projects, designing and developing new webpages, or finding existing bugs and fixing them. The team specialized in many other areas other then simply the University's webpage, contributing to different social media platforms, the team promotes the University to new future Guelph Gryphons, and keeps communication with current students about different campus activities, and encouragement during stressful times.

Job Description

During this work term I had the opportunity to work on one main project with a few side activities and grow my current skills, as well as develope a few new ones on the job.

Main Project

My main project for the University's Communications team started with some research on other University Web Guide pages, determining strengths and flaws with each page, and taking note of the information delivered to the user. From there I began an overhall of the University of Guelph's Web Guide and UI Kit pages. I changed the layout and presentation of the UI Kit, adding new information, sample descriptions, separating different topics to new pages. I even created some new functionality in the javascipt section of the webpage. After the Web Guide and UI Kit were where we wanted it, I took on the task of learning how to use WordPress and then the process of porting the progress on the git repository onto WordPress.

New Skills

During school my web development focused more on the interaction between languages, and the handling of changing information to create a dynamic webpage. During this work term I had the chance to learn a new language, PHP, as well as learn more about different aspects of web development and other web development tools such as Bootstrap and SCSS. Working on the UI Kit sections in particular got me much more familiar with how to create a webpage that is both informative and well designed for the user, but also organised and easy to understand for others looking for simple tools for their pages. Learning how to use WordPress was also a great experience, learning a new way to design and develope a web page that isn't directly in the code.
